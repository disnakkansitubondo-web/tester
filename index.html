Bantuan dan Panduan

Panduan Penggunaan
1. Input Data

Gunakan menu "Input Data" untuk menambahkan data penerima bantuan baru. Pastikan semua field diisi dengan benar.

2. Lihat Data

Menu "Data Penerima" menampilkan semua data yang telah dimasukkan. Anda dapat memfilter data berdasarkan kecamatan, desa, atau status verifikasi.

3. Export Data

Menu "Export Data" memungkinkan Anda mengekspor data ke format Excel atau PDF, serta mencetak laporan.

4. Pengaturan

Menu "Pengaturan" memungkinkan Anda mengubah kode keamanan, menghapus data, atau melakukan backup/restore data.
