Panduan Penggunaan Aplikasi

Panduan Pengguna

  1. Buka menu Input Data
  2. Isi semua field yang wajib diisi (ditandai dengan asterisk (*))
  3. Pastikan NIK terdiri dari 16 digit angka
  4. Pastikan nomor WhatsApp diawali dengan kode negara (contoh: 628123456789)
  5. Klik tombol Simpan Data untuk menyimpan

Pada menu Kelola Data, Anda dapat:

  • Melihat semua data yang telah diinput
  • Menggunakan filter untuk mencari data tertentu
  • Mengedit data dengan klik tombol
  • Menghapus data dengan klik tombol
  • Menghapus beberapa data sekaligus dengan mencentang data dan klik Hapus Data Terpilih
  • Melihat dokumen pendukung dengan klik tombol

Menu Statistik menyediakan visualisasi data dalam bentuk grafik dan tabel.

  1. Pilih tahun yang ingin dianalisis
  2. Pilih jenis bantuan (opsional)
  3. Pilih pengelompokkan data (berdasarkan desa, jenis bantuan, atau bulan)
  4. Klik tombol Generate Statistik untuk menampilkan data

Anda dapat mengekspor data dalam berbagai format:

  • Excel - Format .xlsx untuk dibuka di Microsoft Excel
  • PDF - Format .pdf untuk laporan cetak
  • CSV - Format .csv untuk analisis data di aplikasi lain

Anda juga dapat memfilter data sebelum mengekspor dengan menggunakan filter yang tersedia.

Pada menu Pengaturan, Anda dapat:

  • Melakukan backup data untuk mencegah kehilangan data
  • Restore data dari file backup sebelumnya
  • Mengatur tahun aktif untuk input data
  • Mengaktifkan atau menonaktifkan notifikasi
  • Mengatur jumlah data yang ditampilkan per halaman

Peringatan: Hapus semua data akan menghapus seluruh data yang tersimpan dan tidak dapat dikembalikan.
