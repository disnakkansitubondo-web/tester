Panduan Penggunaan

Panduan Aplikasi BARJAS

  1. Buka menu Input Data
  2. Isi semua field yang wajib diisi (ditandai dengan *)
  3. Pastikan NIK terdiri dari 16 digit angka
  4. Pastikan nomor WhatsApp diawali dengan kode negara (contoh: 628123456789)
  5. Klik tombol Simpan Data untuk menyimpan
  6. Gunakan tombol Auto Fill untuk mengisi data contoh (hanya untuk testing)

  1. Buka menu Kelola Data
  2. Gunakan filter untuk menyaring data berdasarkan desa, jenis bantuan, atau tanggal
  3. Gunakan kotak pencarian untuk mencari data berdasarkan nama, NIK, atau keterangan
  4. Klik header kolom untuk mengurutkan data
  5. Gunakan navigasi halaman di bagian bawah untuk melihat data lebih banyak

  1. Buka menu Cetak Laporan
  2. Tentukan filter data yang ingin dicetak (desa, jenis bantuan, periode)
  3. Pilih jenis laporan (tabel data atau ringkasan)
  4. Klik Preview untuk melihat hasil sebelum mencetak
  5. Klik Cetak untuk mencetak langsung atau Export PDF untuk menyimpan sebagai PDF
  6. Pastikan browser tidak memblokir popup untuk fungsi cetak

  1. Buka menu Export Data
  2. Tentukan filter data yang ingin diexport
  3. Pilih format export (Excel, CSV, atau JSON)
  4. Klik tombol Export Data
  5. File akan otomatis diunduh ke perangkat Anda
  6. Riwayat export dapat dilihat di tabel bawah

  1. Buka menu Pengaturan
  2. Untuk mengubah kode keamanan, masukkan kode saat ini kemudian kode baru
  3. Klik Simpan Perubahan untuk menyimpan kode baru
  4. Gunakan fitur Backup Data untuk menyimpan cadangan data
  5. Gunakan fitur Restore Data untuk memulihkan data dari cadangan
  6. Hati-hati dengan fitur Reset Data karena akan menghapus semua data
Pertanyaan Umum

Hubungi developer di +62 856-4770-9114 untuk reset kode keamanan. Pastikan Anda adalah admin yang berwenang.

Pastikan semua field wajib diisi. Periksa koneksi internet jika aplikasi disimpan di server. Jika menggunakan penyimpanan lokal, pastikan browser mendukung dan tidak dalam mode penyamaran.

Pada menu Cetak Laporan, gunakan filter "Desa" untuk memilih desa tertentu. Anda juga bisa mengkombinasikan dengan filter jenis bantuan dan periode.

Data disimpan secara lokal di browser Anda. Untuk keamanan ekstra, lakukan backup data secara berkala dan jangan bagikan kode keamanan kepada orang lain.
