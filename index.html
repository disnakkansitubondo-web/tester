Panduan Penggunaan Aplikasi

  1. Buka menu Input Data
  2. Isi semua field yang wajib diisi (ditandai dengan asterisk)
  3. Pastikan nomor WhatsApp dalam format yang benar
  4. Pilih desa dari dropdown atau ketik untuk mencari
  5. Pilih jenis bantuan atau ketik jenis lainnya jika tidak ada dalam pilihan
  6. Generate kode validasi atau masukkan kode manual
  7. Klik Simpan Data untuk menyimpan

Pada menu Data Penerima, Anda dapat:

  • Melihat semua data yang telah diinput
  • Menggunakan filter berdasarkan desa, status, atau tanggal
  • Mencari data dengan kata kunci tertentu
  • Mengatur jumlah data yang ditampilkan per halaman
  • Mengurutkan data dengan klik pada header kolom

Anda dapat mengekspor data dalam format Excel atau PDF:

  • Pilih filter yang diinginkan (jika perlu)
  • Klik tombol Excel untuk export data dalam format XLSX
  • Klik tombol PDF untuk export data dalam format PDF
  • Klik tombol Cetak untuk mencetak data langsung
  • Untuk mencetak data individual, gunakan tombol cetak pada kolom aksi

Menu Laporan menyediakan visualisasi data:

  • Grafik batang menunjukkan distribusi per desa
  • Grafik pie menunjukkan distribusi per status
  • Tabel statistik menunjukkan rincian per desa
  • Data dapat diekspor untuk analisis lebih lanjut

Pada menu Pengaturan, Anda dapat:

  • Mengganti kode keamanan aplikasi
  • Mengatur opsi tampilan
  • Mereset data aplikasi (hati-hati, data akan hilang)
  • Mencadangkan dan memulihkan data
