Bantuan & Panduan

Panduan Penggunaan

  1. Buka menu Input Data
  2. Isi semua field yang wajib diisi (ditandai dengan *)
  3. Untuk nomor WhatsApp, masukkan dengan format 08xx, akan otomatis dikonversi ke format 62
  4. Untuk link Google Drive, pastikan link sudah dalam format yang dapat diakses
  5. Klik tombol Simpan Data untuk menyimpan

Pada menu Data Bantuan, Anda dapat:

  • Melihat semua data yang telah diinput
  • Menggunakan filter untuk menyaring data berdasarkan desa, jenis bantuan, atau periode tanggal
  • Mengklik nomor WhatsApp untuk langsung terhubung ke WhatsApp
  • Mengklik link Google Drive untuk melihat dokumentasi
  • Mengedit atau menghapus data dengan tombol aksi di kolom terakhir

Pada menu Cetak Laporan, Anda dapat:

  • Mencetak laporan dalam format PDF
  • Mencetak langsung ke printer
  • Melakukan pratinjau sebelum mencetak
  • Menggunakan filter untuk mencetak data tertentu

Catatan: Pastikan printer sudah terhubung dan siap digunakan untuk cetak langsung.

Pada menu Ekspor Data, Anda dapat mengekspor data ke berbagai format:

  • Excel: Format .xlsx yang dapat dibuka dengan Microsoft Excel
  • CSV: Format teks dengan pemisah koma, kompatibel dengan berbagai aplikasi
  • JSON: Format untuk pertukaran data, berguna untuk developer
  • PDF: Format dokumen portabel yang mudah dibagikan

Gunakan filter untuk mengekspor data tertentu sesuai kebutuhan.

Informasi Penting

Aplikasi BARJAS menyimpan data secara lokal di browser Anda.

Pastikan untuk melakukan backup data secara berkala untuk menghindari kehilangan data.

Jika mengalami kendala, hubungi developer melalui informasi kontak yang tersedia.
