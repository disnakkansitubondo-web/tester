Panduan Penggunaan

Bantuan dan Dukungan
1. Cara Menambahkan Surat

Untuk menambahkan surat masuk atau keluar, pilih menu yang sesuai, isi form yang disediakan, lalu klik tombol "Simpan".

2. Validasi Surat

Gunakan tombol "Generate" untuk membuat kode validasi unik. Pastikan untuk mengisi nama petugas validasi.

3. Pencarian Surat

Gunakan fitur pencarian di atas tabel untuk mencari surat berdasarkan nomor, pengirim, perihal, atau lainnya.

4. Backup dan Restore

Selalu lakukan backup data secara berkala. Fitur restore dapat digunakan untuk mengembalikan data dari backup.

5. Mengubah Password

Ubah password default untuk keamanan yang lebih baik melalui menu Pengaturan.

Untuk bantuan lebih lanjut, hubungi tim IT di it-support@lenterakarya.situbondo.id atau telepon (0338) 671234.
