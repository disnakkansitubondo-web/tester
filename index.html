Bantuan dan Panduan

Panduan Penggunaan

  1. Buka menu Input Data
  2. Isi semua field yang wajib diisi (ditandai dengan *)
  3. Klik tombol Simpan Data
  4. Data akan tersimpan dan muncul di tabel

  1. Buka menu Ekspor Data
  2. Pilih opsi Cetak Laporan
  3. Pilih format laporan yang diinginkan
  4. Klik tombol Cetak Laporan
  5. Browser akan membuka jendela cetak

  1. Buka menu Kelola Data
  2. Gunakan dropdown filter untuk memfilter berdasarkan desa, jenis bantuan, atau status
  3. Gunakan kotak pencarian untuk mencari data tertentu
  4. Data akan otomatis tersaring sesuai kriteria
