Bantuan & Panduan

Panduan Penggunaan

  1. Buka menu Input Data
  2. Isi semua field yang wajib diisi (ditandai dengan asterisk (*))
  3. Pastikan NIK terdiri dari 16 digit angka
  4. Pastikan nomor telepon diawali dengan kode negara (+62)
  5. Klik tombol Simpan Data untuk menyimpan data

Pada menu Kelola Data, Anda dapat:

  • Melihat semua data penerima bantuan
  • Menggunakan filter untuk menyaring data berdasarkan desa, jenis bantuan, atau tahun
  • Mencari data tertentu menggunakan kolom pencarian
  • Mengedit data dengan klik tombol Edit pada kolom aksi
  • Menghapus data dengan klik tombol Hapus pada kolom aksi

Pada menu Ekspor Data, Anda dapat:

  • Mengekspor data ke format PDF untuk keperluan laporan
  • Mengekspor data ke format Excel untuk analisis lebih lanjut
  • Mencetak laporan langsung dari browser
  • Menggunakan filter untuk mengekspor data tertentu saja

Pada menu Statistik, Anda dapat:

  • Melihat grafik distribusi bantuan per desa
  • Melihat grafik distribusi jenis bantuan
  • Melihat tabel statistik lengkap untuk analisis data
