Bantuan & Panduan

Panduan Penggunaan

  1. Buka menu Input Data
  2. Isi semua field yang wajib diisi (ditandai dengan *)
  3. Klik tombol Simpan Data untuk menyimpan
  4. Data akan tersimpan dan dapat dilihat di menu Data Bantuan

  1. Buka menu Data Bantuan
  2. Gunakan filter yang tersedia (Desa, Jenis, Tanggal)
  3. Klik tombol Terapkan Filter untuk memfilter data
  4. Klik tombol Reset Filter untuk menampilkan semua data

  1. Buka menu Data Bantuan
  2. Gunakan filter jika ingin mengekspor data tertentu
  3. Klik tombol Excel untuk export ke format Excel
  4. Klik tombol PDF untuk export ke format PDF
  5. Klik tombol Cetak untuk mencetak data
Troubleshooting

Pastikan semua field wajib sudah diisi dengan benar. Periksa juga apakah browser mendukung penyimpanan lokal.

Pastikan format tanggal yang dimasukkan sudah benar. Coba reset filter dan terapkan kembali.

Hubungi administrator untuk reset kode keamanan. Informasi kontak administrator tersedia di bagian bawah sidebar.
