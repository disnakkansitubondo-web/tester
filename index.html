Panduan Penggunaan

Cara Menggunakan Aplikasi

  1. Buka menu Input Data
  2. Isi semua field yang diperlukan (nama, desa, jenis bantuan, jumlah, tanggal)
  3. Field NIK dan WhatsApp bersifat opsional
  4. Untuk link Google Drive, masukkan URL lengkap dokumen yang ingin dihubungkan
  5. Klik Simpan Data untuk menyimpan

  1. Buka menu Data Bantuan
  2. Gunakan kotak pencarian untuk mencari data berdasarkan nama
  3. Gunakan filter desa, jenis bantuan, atau tanggal untuk mempersempit hasil
  4. Klik pada header kolom untuk mengurutkan data
  5. Gunakan navigasi halaman untuk melihat data yang banyak

  1. Buka menu Cetak Laporan
  2. Tentukan filter yang diinginkan (tanggal, jenis bantuan, desa)
  3. Klik Pratinjau Laporan untuk melihat hasilnya terlebih dahulu
  4. Klik Cetak Sekarang untuk langsung mencetak ke printer
  5. Klik Cetak PDF untuk menyimpan laporan dalam format PDF
Ekspor Data

  1. Buka menu Ekspor Data
  2. Tentukan filter data yang ingin diekspor
  3. Pilih format Excel (.xlsx)
  4. Tentukan kolom-kolom yang ingin disertakan
  5. Klik Ekspor Data untuk mengunduh file

  1. Buka menu Ekspor Data

Troubleshooting
Data tidak tersimpan? Pastikan semua field wajib diisi dan koneksi internet stabil.
Laporan tidak tercetak? Pastikan printer terhubung dan driver sudah terinstall.
Ekspor data gagal? Pastikan tidak ada data yang kosong pada kolom yang dipilih.
