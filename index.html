Bantuan dan Panduan Penggunaan

Panduan Penggunaan Aplikasi

  1. Pergi ke menu Input Data
  2. Isi semua field yang wajib diisi (ditandai dengan *)
  3. Pastikan NIK terdiri dari 16 digit angka
  4. Pastikan nomor WhatsApp diawali dengan kode negara (contoh: 628123456789)
  5. Pilih kecamatan terlebih dahulu, kemudian pilih desa
  6. Pilih jenis bantuan sosial yang diberikan
  7. Klik tombol Simpan Data untuk menyimpan

  1. Pergi ke menu Kelola Data
  2. Gunakan filter untuk menyaring data berdasarkan kecamatan, desa, atau jenis bansos
  3. Gunakan kotak pencarian untuk mencari data berdasarkan nama, NIK, atau alamat
  4. Klik tombol Edit untuk mengubah data yang sudah ada
  5. Klik tombol Hapus untuk menghapus data (data yang dihapus tidak dapat dikembalikan)
  6. Gunakan navigasi halaman untuk melihat data yang lebih banyak

  1. Pergi ke menu Statistik untuk melihat grafik dan tabel statistik
  2. Gunakan filter untuk menyesuaikan tampilan statistik
  3. Pergi ke menu Ekspor Data untuk mengekspor data ke format Excel, PDF, atau CSV
  4. Pilih format, rentang tanggal, dan filter data yang ingin diekspor
  5. Klik tombol Ekspor Data untuk mengunduh file
