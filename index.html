Bantuan dan Panduan Penggunaan

Panduan Penggunaan

  1. Buka menu Tambah Data
  2. Isi semua field yang wajib diisi (ditandai dengan *)
  3. Klik tombol Simpan Data untuk menyimpan
  4. Data akan muncul di tabel Data Penerima

  1. Download template Excel dengan mengklik link Download template
  2. Isi data sesuai format template
  3. Buka menu Import Data
  4. Upload file Excel yang telah diisi
  5. Klik tombol Import Data untuk memproses

Anda dapat mengekspor data dalam berbagai format:

  • Excel: Untuk pengolahan data lebih lanjut
  • PDF: Untuk dokumentasi dan laporan
  • CSV: Untuk kompatibilitas dengan berbagai aplikasi
  • JSON: Untuk backup dan transfer data

Pilih format yang diinginkan di menu Export Data

Backup Data:

  1. Buka menu Backup Data
  2. Klik tombol Buat Backup
  3. File backup akan diunduh secara otomatis

Restore Data:

  1. Buka menu Backup Data
  2. Pilih file backup yang ingin dipulihkan
  3. Klik tombol Restore Data
  4. Data akan diganti dengan data dari file backup
Pemecahan Masalah

Pastikan Anda telah mengisi semua field yang wajib diisi. Field wajib ditandai dengan tanda asterisk (*) atau memiliki pesan error jika dikosongkan.

Pastikan file Excel yang diupload sesuai dengan format template. Periksa juga apakah tidak ada kolom yang kosong pada data yang wajib diisi.

Coba refresh halaman browser. Jika masalah berlanjut, coba hapus cache browser atau gunakan browser yang berbeda.
