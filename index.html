Panduan Penggunaan Aplikasi

Panduan Lengkap

  1. Buka menu Input Data
  2. Isi semua field yang diperlukan (nama, alamat, desa, dll)
  3. Untuk nomor WhatsApp, masukkan tanpa kode negara (+62)
  4. Pilih jenis bantuan atau ketik jenis lainnya jika tidak ada dalam pilihan
  5. Masukkan nilai bantuan dalam Rupiah
  6. Tambahkan link Google Drive jika diperlukan (opsional)
  7. Klik tombol Simpan Data untuk menyimpan

  1. Buka menu Data Bantuan
  2. Gunakan filter di atas tabel untuk menyaring data berdasarkan desa, jenis bantuan, atau tanggal
  3. Gunakan kotak pencarian untuk mencari data berdasarkan kata kunci
  4. Gunakan tombol navigasi untuk berpindah halaman jika data banyak
  5. Klik tombol untuk melihat detail data
  6. Klik tombol untuk mengedit data
  7. Klik tombol untuk menghapus data

  1. Buka menu Laporan
  2. Untuk membuat laporan PDF, pilih filter yang diinginkan lalu klik Unduh Laporan PDF
  3. Untuk ekspor Excel, pilih filter yang diinginkan lalu klik Ekspor ke Excel
  4. Gunakan fitur pratinjau untuk melihat data sebelum mencetak
  5. Klik Cetak Pratinjau untuk mencetak data yang ditampilkan

  • Gunakan tombol Auto Fill untuk mengisi form dengan data contoh (hanya untuk testing)
  • Simpan link Google Drive untuk dokumen pendukung seperti foto atau surat
  • Gunakan filter yang tersedia untuk analisis data yang lebih spesifik
  • Ekspor data secara berkala untuk backup
  • Hubungi developer jika menemukan kendala atau membutuhkan fitur tambahan
