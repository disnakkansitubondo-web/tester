Bantuan & Panduan

Panduan Penggunaan

  1. Buka menu Input Data
  2. Isi semua field yang wajib diisi (nama, desa, jenis bantuan, jumlah, tanggal)
  3. Field WhatsApp dan Google Drive opsional
  4. Klik tombol Simpan Data untuk menyimpan
  5. Gunakan tombol Auto Fill untuk mengisi data contoh

  1. Buka menu Data Bantuan untuk melihat semua data
  2. Gunakan filter untuk menyaring data berdasarkan desa, jenis, atau tanggal
  3. Gunakan kolom pencarian untuk mencari data tertentu
  4. Buka menu Statistik untuk melihat grafik dan analisis data
  5. Gunakan filter tahun dan bulan pada menu statistik untuk analisis periode tertentu

  1. Buka menu Export Data
  2. Pilih format export (PDF, Excel, atau Cetak Langsung)
  3. Tentukan rentang data yang akan diexport (semua, terpilih, atau tersaring)
  4. Klik tombol export sesuai format yang diinginkan
  5. Untuk mencetak, pastikan printer sudah terhubung dan siap

  1. Buka menu Pengaturan
  2. Ubah kode keamanan aplikasi di bagian Pengaturan Keamanan
  3. Backup data secara berkala untuk mencegah kehilangan data
  4. Ganti tema dan ukuran font sesuai preferensi di bagian Tampilan
  5. Pantau penggunaan penyimpanan di bagian Informasi Aplikasi
Troubleshooting

Pastikan semua field wajib diisi dan browser mendukung penyimpanan lokal. Coba bersihkan cache browser atau gunakan browser yang berbeda.

Pastikan koneksi internet aktif karena library grafik diambil dari CDN. Jika tidak ada internet, beberapa fitur grafik mungkin tidak berfungsi.

Jika data sudah sangat banyak, coba hapus data yang sudah tidak diperlukan atau lakukan backup lalu hapus semua data untuk memulai baru.
