Bantuan & Panduan

Panduan Penggunaan

  1. Buka menu Input Data
  2. Isi semua field yang wajib diisi (ditandai dengan *)
  3. Klik tombol Simpan Data untuk menyimpan
  4. Data akan langsung tersimpan dan muncul di tabel

  1. Buka menu Kelola Data
  2. Gunakan filter untuk menyaring data jika diperlukan
  3. Klik tombol Edit untuk mengubah data
  4. Klik tombol Hapus untuk menghapus data
  5. Gunakan fitur pencarian untuk mencari data tertentu

  1. Buka menu Ekspor Data
  2. Pilih format ekspor (PDF, Excel, atau Cetak)
  3. Atur opsi ekspor sesuai kebutuhan
  4. Klik tombol ekspor untuk mengunduh atau mencetak
Troubleshooting

Pastikan:

  • Semua field wajib sudah diisi
  • Koneksi internet stabil (jika menggunakan penyimpanan cloud)
  • Browser mendukung fitur penyimpanan lokal

Coba langkah berikut:

  • Klik tombol Reset Filter kemudian terapkan filter lagi
  • Pastikan format pencarian sesuai dengan data yang dicari
  • Refresh halaman jika diperlukan

Beberapa hal yang dapat dilakukan:

  • Bersihkan cache browser
  • Kurangi jumlah data yang disimpan (jika terlalu banyak)
  • Gunakan browser terbaru
  • Pastikan spesifikasi device memadai
