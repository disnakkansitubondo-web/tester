Bantuan & Panduan Penggunaan

Panduan Penggunaan Aplikasi

  1. Buka menu Input Data dari sidebar navigasi
  2. Isi semua field yang wajib diisi (ditandai dengan asterisk)
  3. Pastikan NIK dan No. KK terdiri dari 16 digit angka
  4. Pilih Kecamatan terlebih dahulu, kemudian pilih Desa dari dropdown
  5. Pilih Jenis Bansos, Tahun, dan Periode pemberian bantuan
  6. Klik tombol Generate Kode untuk membuat kode validasi otomatis
  7. Klik Simpan Data untuk menyimpan data ke sistem

Pada menu Kelola Data, Anda dapat:

  • Melihat semua data penerima bantuan dalam bentuk tabel
  • Menggunakan filter untuk menyaring data berdasarkan Kecamatan, Desa, Jenis Bansos, atau Tahun
  • Mencari data tertentu menggunakan kolom pencarian
  • Mengurutkan data berdasarkan kolom tertentu
  • Mengedit atau menghapus data dengan klik tombol aksi pada setiap baris data
  • Mencetak data atau mengekspor ke format Excel/PDF

Pada menu Ekspor Data, Anda dapat:

  • Mengekspor data ke format Excel dengan memilih kolom tertentu yang ingin disertakan
  • Mengekspor data ke format PDF dengan berbagai template laporan
  • Melakukan backup seluruh data aplikasi untuk keperluan restore di kemudian hari
  • Restore data dari file backup sebelumnya

Catatan: Pastikan untuk menyimpan file backup di lokasi yang aman.

Pada menu Pengaturan, Anda dapat:

  • Mengubah judul dan subjudul aplikasi
  • Mengatur tahun default untuk input data
  • Mengatur jumlah item per halaman pada tabel data
  • Mengaktifkan atau menonaktifkan fitur auto save
  • Mengelola data (hapus semua data, ekspor, atau impor data)
  • Mengubah kode keamanan aplikasi
Troubleshooting

Jika data tidak tersimpan, pastikan:

  • Semua field wajib sudah diisi dengan benar
  • NIK dan No. KK terdiri dari 16 digit angka
  • Koneksi internet stabil (jika menggunakan penyimpanan cloud)
  • Penyimpanan lokal browser tidak penuh

Jika aplikasi berjalan lambat:

  • Pastikan browser Anda telah diperbarui ke versi terbaru
  • Coba tutup tab browser lain yang tidak diperlukan
  • Bersihkan cache dan cookies browser
  • Kurangi jumlah data yang disimpan (hapus data yang tidak perlu)

Jika Anda lupa kode keamanan aplikasi:

  • Hubungi administrator sistem
  • Jika Anda adalah administrator, kode disimpan di localStorage browser dengan key securityCode
  • Anda dapat reset aplikasi dengan menghapus semua data localStorage (termasuk kode keamanan)
Hati-hati saat reset aplikasi, karena semua data akan terhapus.

Deskripsi:

Aplikasi Sistem Bantuan Terpadu untuk mengelola data penerima bantuan sosial di Kabupaten Situbondo.

Fitur Utama:
  • Input data penerima bantuan
  • Kelola data dengan filter dan pencarian
  • Statistik dan visualisasi data
  • Ekspor data ke Excel dan PDF
  • Backup dan restore data
  • Keamanan dengan kode akses
Teknologi:
  • HTML5, CSS3, JavaScript
  • Bootstrap 5.3.0
  • Font Awesome 6.4.0
  • Chart.js untuk grafik
  • jsPDF untuk ekspor PDF
  • SheetJS untuk ekspor Excel
