Pengumuman Penting

Aplikasi BARJAS v2.1.0 telah dirilis dengan perbaikan fitur input data dan penambahan opsi ekspor data.

Panduan Singkat
  1. Gunakan menu Input Data untuk menambahkan data bantuan baru
  2. Gunakan menu Kelola Data untuk melihat, mengedit, atau menghapus data
  3. Gunakan menu Link Dokumen untuk mengelola tautan dokumen pendukung
  4. Gunakan menu Statistik untuk melihat grafik dan laporan data

Bantuan dan Panduan

Panduan penggunaan aplikasi dan informasi bantuan.

Panduan Penggunaan

  1. Buka menu Input Data
  2. Isi semua field yang wajib diisi (ditandai dengan *)
  3. Gunakan tombol Gunakan GPS untuk mendapatkan koordinat lokasi secara otomatis
  4. Klik tombol Simpan Data untuk menyimpan

Pada menu Kelola Data, Anda dapat:

  • Melihat semua data yang telah disimpan
  • Menggunakan filter untuk menyaring data
  • Mengedit data dengan klik tombol
  • Menghapus data dengan klik tombol
  • Mengekspor data ke Excel atau PDF
  • Mencetak data yang ditampilkan

  1. Buka menu Link Dokumen
  2. Isi field Label Tautan dan URL Tautan
  3. Klik tombol Tambah Tautan
  4. Tautan akan tersimpan dan dapat diakses melalui tabel
