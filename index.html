Panduan Penggunaan

1 Input Surat Masuk

Untuk menambahkan surat masuk baru, klik menu Input Surat Masuk dan isi formulir yang tersedia. Pastikan semua informasi diisi dengan benar sebelum menyimpan.

2 Input Surat Keluar

Untuk menambahkan surat keluar baru, klik menu Input Surat Keluar dan isi formulir yang tersedia. Pastikan semua informasi diisi dengan benar sebelum menyimpan.

3 Melihat Data Surat

Untuk melihat data surat yang telah disimpan, gunakan menu Data Surat Masuk atau Data Surat Keluar. Anda dapat mencari, menyaring, dan mengurutkan data sesuai kebutuhan.

4 Ekspor Data

Anda dapat mengekspor data surat ke format Excel atau PDF dengan menggunakan tombol ekspor yang tersedia di halaman data surat.

5 Mengubah Password

Untuk mengubah password aplikasi, gunakan menu Pengaturan dan pilih opsi Ubah Password.

6 Backup dan Restore

Selalu lakukan backup data secara berkala melalui menu Pengaturan untuk mencegah kehilangan data.

BARJAS - Sistem Bantuan Surat Masuk & Keluar

Aplikasi ini dirancang untuk membantu pengelolaan surat masuk dan surat keluar dengan fitur yang lengkap dan mudah digunakan.

Fitur Utama:
  • Input dan pengelolaan surat masuk
  • Input dan pengelolaan surat keluar
  • Pencarian dan penyaringan data
  • Ekspor data ke Excel dan PDF
  • Backup dan restore data
  • Antarmuka yang responsif
