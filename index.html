Informasi Aplikasi

SISTEM MANAJEMEN SURAT

Aplikasi ini dirancang khusus untuk Dinas Peternakan dan Perikanan Kabupaten Situbondo untuk mengelola surat masuk dan surat keluar secara digital.

Fitur Utama:
  • Input dan kelola surat masuk dan keluar
  • Pencarian dan filter data surat
  • Ekspor data ke Excel dan PDF
  • Integrasi dengan Google Drive
  • Backup dan restore data
  • Keamanan data dengan password
Versi Aplikasi: 2.0.0

Tanggal Rilis: 1 Januari 2025

Teknologi:
  • HTML5, CSS3, JavaScript
  • Bootstrap 5.3.0
  • Font Awesome 6.4.0
  • SheetJS (xlsx)
  • jsPDF
