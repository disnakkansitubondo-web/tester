Bantuan & Panduan

Panduan Penggunaan

  1. Buka menu Input Data
  2. Isi semua field yang diperlukan (Nama, Desa, Jenis Bantuan, dll)
  3. Klik tombol Generate untuk membuat kode validasi
  4. Salin kode yang dihasilkan dan tempel di field Kode Validasi
  5. Klik tombol Simpan Data untuk menyimpan

  1. Buka menu Ekspor Data
  2. Pilih opsi Cetak Laporan
  3. Tentukan jenis laporan yang diinginkan
  4. Atur filter sesuai kebutuhan
  5. Klik tombol Cetak Laporan
  6. Browser akan membuka jendela cetak

  1. Buka menu Pengaturan
  2. Pergi ke bagian Kelola Data
  3. Klik tombol Backup Data
  4. File backup akan diunduh ke perangkat Anda
  5. Simpan file backup di lokasi yang aman

Pemecahan Masalah
Data tidak tersimpan?

Pastikan semua field wajib diisi dan kode validasi dimasukkan dengan benar.

Laporan tidak tercetak?

Pastikan pop-up blocker dinonaktifkan untuk situs ini.

Aplikasi lambat?

Bersihkan cache browser atau kurangi jumlah data yang disimpan.
