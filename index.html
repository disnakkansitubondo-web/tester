Panduan Penggunaan

  1. Buka menu Input Data
  2. Isi semua field yang wajib diisi (ditandai dengan asterisk)
  3. Pastikan data yang dimasukkan sudah benar
  4. Klik tombol Simpan Data untuk menyimpan
  5. Data akan tersimpan dan dapat dilihat di menu Data Bantuan

Di menu Data Bantuan, Anda dapat memfilter data berdasarkan:

  • Desa - Pilih desa tertentu untuk melihat data dari desa tersebut
  • Jenis Bantuan - Filter berdasarkan jenis bantuan
  • Tanggal - Tampilkan data pada tanggal tertentu
  • Pencarian - Cari data berdasarkan nama atau alamat

Gunakan tombol Terapkan Filter untuk menerapkan filter yang dipilih, dan Reset untuk mengatur ulang filter.

Di menu Laporan, Anda dapat membuat laporan dalam format PDF atau Excel:

  1. Pilih rentang tanggal, jenis bantuan, dan desa (opsional)
  2. Klik tombol Buat Laporan PDF atau Ekspor ke Excel
  3. File akan diunduh secara otomatis

Anda juga dapat melihat pratinjau data sebelum mencetak dengan menekan tombol Tampilkan Pratinjau.

Di menu Pengaturan, Anda dapat:

  • Mengganti kode keamanan aplikasi
  • Mengatur daftar desa yang tersedia
  • Mengatur jenis bantuan yang tersedia
  • Mencadangkan dan memulihkan data

Peringatan: Hanya administrator yang dapat mengakses pengaturan ini.
