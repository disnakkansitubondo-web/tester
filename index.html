Bantuan dan Panduan

Panduan Penggunaan

  1. Buka menu Tambah Data
  2. Isi semua field yang wajib diisi (ditandai dengan *)
  3. Klik Simpan Data untuk menyimpan
  4. Gunakan Preview untuk melihat data sebelum disimpan

  1. Download template Excel terlebih dahulu
  2. Isi data sesuai dengan format template
  3. Buka menu Import Data
  4. Unggah file Excel yang telah diisi
  5. Klik Import Data untuk memproses

  1. Buka menu Export Data
  2. Pilih format export (Excel atau PDF)
  3. Atur opsi export sesuai kebutuhan
  4. Klik tombol export yang diinginkan
  5. Tunggu hingga proses export selesai
Pemecahan Masalah

Pastikan semua field wajib telah diisi dan koneksi internet stabil.

Pastikan format file Excel sesuai dengan template dan tidak ada data yang kosong pada kolom wajib.
