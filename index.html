Panduan Penggunaan

Panduan Aplikasi BARJAS

  1. Buka menu Input Data
  2. Isi semua field yang wajib diisi (ditandai dengan asterisk (*))
  3. Pilih kecamatan terlebih dahulu, kemudian pilih desa
  4. Pilih jenis bantuan dan tanggal penerimaan
  5. Klik tombol Simpan Data untuk menyimpan
  6. Data akan tersimpan di browser lokal perangkat Anda

  1. Buka menu Kelola Data
  2. Gunakan filter di atas tabel untuk menyaring data
  3. Anda dapat filter berdasarkan kecamatan, desa, jenis bantuan, atau tahun
  4. Klik tombol Terapkan Filter untuk menerapkan filter
  5. Gunakan kotak pencarian untuk mencari data tertentu

  1. Buka menu Ekspor Data
  2. Pilih format ekspor: Excel, PDF, atau Cetak Laporan
  3. Untuk Excel: Klik tombol Ekspor ke Excel
  4. Untuk PDF: Klik tombol Ekspor ke PDF
  5. Untuk mencetak: Pilih format laporan dan tahun, lalu klik Cetak Laporan

  1. Buka menu Pengaturan
  2. Untuk backup: Klik tombol Backup Data untuk mengunduh file backup
  3. Untuk restore: Klik tombol Restore Data dan pilih file backup sebelumnya
  4. Hati-hati: Hapus Semua Data akan menghapus semua data permanen

  1. Buka menu Pengaturan
  2. Masukkan kode keamanan baru di field Ganti Kode Keamanan
  3. Klik tombol Simpan Kode Baru
  4. Kode akan digunakan untuk login ke aplikasi berikutnya
  5. Jangan lupakan kode Anda karena tidak dapat dikembalikan
Pertanyaan Umum

Data disimpan di local storage browser perangkat Anda. Artinya data hanya tersimpan di browser yang Anda gunakan dan tidak terkirim ke server manapun.

Jika lupa kode keamanan, Anda perlu menghapus data aplikasi dari browser dan memulai kembali. Seluruh data akan hilang jika melakukan reset, jadi pastikan sudah melakukan backup data sebelumnya.

Gunakan filter di menu Kelola Data untuk menyaring data yang ingin dicetak. Setelah data tersaring, buka menu Ekspor Data dan pilih opsi Cetak Laporan.

Data Anda aman selama tidak ada orang lain yang mengakses perangkat dan browser Anda. Untuk keamanan tambahan, gunakan fitur kode keamanan dan jangan berikan kode kepada orang lain.
