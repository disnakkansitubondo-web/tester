Bantuan dan Panduan

Panduan Penggunaan

  1. Buka menu Input Data
  2. Isi semua field yang wajib diisi (ditandai dengan *)
  3. Klik tombol Simpan Data untuk menyimpan
  4. Data akan tersimpan dan tampil di menu Data Bantuan

  1. Buka menu Laporan
  2. Pilih tahun dan jenis laporan yang diinginkan
  3. Klik tombol Generate Laporan
  4. Laporan akan dibuat dalam format PDF

  1. Buka menu Ekspor Data
  2. Klik tombol Cadangkan Data
  3. File JSON akan diunduh ke perangkat Anda
  4. Simpan file ini di lokasi yang aman untuk pemulihan data
